# Refugio Seguro

Sitio web independiente inspirado en una estructura editorial tipo portal.

## Incluye

- gran hero de apertura
- bloques de guía
- tabla comparativa
- gráficos dinámicos
- sección de video destacada
- módulos de exploración
- pie de página con enlaces

## Archivos

- `index.html`
- `styles.css`
- `script.js`

## Nota

La referencia estructural fue `starwars.com`, pero el contenido, la identidad y el diseño fueron adaptados para mantener un proyecto original sobre prevención y apoyo.
